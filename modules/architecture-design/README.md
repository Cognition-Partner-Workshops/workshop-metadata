- Architecture-focused audiences (solution architects, tech leads, principal engineers)
- Workshops showing Devin's ability to analyze system structure and produce architectural artifacts
- ADRs and API Design Review are particularly valuable for teams establishing governance practices
- Dependency Graph Analysis helps teams understand coupling and plan modernization efforts
- Pairs well with [Migration & Modernization](../migration-modernization/) for architecture-led modernization workshops
- Code Refactoring & Tech Debt is a good entry point for teams with existing technical debt backlogs
