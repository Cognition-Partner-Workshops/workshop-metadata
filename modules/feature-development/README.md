## When to Use This Category

- Developer-focused audiences who want to see Devin as a coding partner

- Good mix of difficulty levels for diverse audiences
- D4 and D5 have specific, verifiable bugs — good for "see the fix live" demos
- D1 and D2 showcase Devin's planning and reasoning, not just code generation
