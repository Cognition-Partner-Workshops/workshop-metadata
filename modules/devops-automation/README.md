# Category E: DevOps, Automation, and Controllers

Challenges focused on CI/CD, observability, PR review, and automated failure resolution.

| ID | Challenge | Primary Repos | Difficulty | Time Est. |
|----|-----------|---------------|-----------|-----------|
| E1 | [CI/CD](da-cicd.md) | Any | Intermediate | 60 min |
| E2 | [Observability](da-observability.md) | Any | Intermediate | 45 min |
| E3 | [PR Review](da-pr-review.md) | Any | Beginner–Intermediate | 30 min |
| E4 | [CI Failure Resolution](da-ci-failure-resolution.md) | Any | Advanced | 60 min |

## When to Use This Category

- DevOps / Platform Engineering audiences
- Workshops demonstrating Devin as an automation agent (not just a coder)
- E3 (PR Review) is a great low-risk entry point — Devin reads and comments, no code changes
- E4 (CI Failure Resolution) is the most advanced — shows Devin as an autonomous remediation agent
- Pairs well with Category B (Security) for a "DevSecOps" themed workshop
