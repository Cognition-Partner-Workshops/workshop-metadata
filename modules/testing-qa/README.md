## When to Use This Category

- QA-focused audiences — Linting and Unit Testing are approachable entry points
- Workshops showing Devin's ability to generate meaningful tests, not just boilerplate
- Contract Testing and Cross-Service Integration Testing pair well with [Migration & Modernization](../migration-modernization/) for microservice decomposition workshops
- Performance Testing and Load Testing cover non-functional requirements often overlooked in workshops
- Mutation Testing showcases advanced test quality analysis
- Pairs well with [Security](../security/) for a "quality + security" half-day workshop
